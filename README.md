# Express TypeScript Starter

간단한 **Express + TypeScript + Prettier** 설정 템플릿

## ✅ Features

- ⚙️ Express 서버 기본 세팅
- 📦 TypeScript 개발 환경 구성
- 💅 Prettier 코드 포매터 적용


## 🚀 Getting Started

### 1. 의존성 설치

```bash
yarn install
```
